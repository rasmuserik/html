# JavaScript platform

Features:
- TypedArrays
- SIMD: mapPar etc.
- threads: WebWorkers
- network: webrtc, websocket, ajax
- graphics: canvas, webgl, dom(inkl. svg)
- input: touch, webrtc-video-capture, geo, accel
- performance: (hidden classes, generational GC, )

- Java gwt
